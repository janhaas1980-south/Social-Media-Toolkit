## 🔍 核心痛点：为什么你的 4K 直播会卡顿？
经过曼谷实验室抓包分析，90% 的卡顿并非终端性能不足，而是因为使用了 **数据中心 IP (机房 IP)**。这类 IP 信誉分极低，极易触发服务端的 TLS 协议重置 (RST)。
* **本方案解法**：通过切换至 **住宅原生 ISP (家庭住宅 IP)**，彻底避开机房风控特征，实现白名单级的连接权重。

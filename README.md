# 🍎 美国 App Store 公共账号共享计划 (US-AppStore-Public-Pool)

本项目旨在为全球开发者及用户提供免费、稳定的美国区域 Apple ID，主要用于下载免费应用、日常维护及技术测试。





### ⚠️ 核心规则（必读）
- **严禁登录 iCloud**：仅允许在 App Store 中登录。若因操作不当导致锁机，本仓库概不负责。
- **请勿尝试修改密码**：账号为多人共享，修改密码会导致所有人无法使用且账号被锁定。
- **用完即退**：下载完成后请立即退出，方便他人使用。
